Classes

The biggest thing to remember with classes is that they start with a . in CSS. For example, div is an element selector, but .first is a class selector. You don't need to use the . when you're applying the class in HTML.

The box model

Link to video - https://vimeo.com/110067455

Link to code - http://jsbin.com/xadeju/1/edit?html,css,output

The elements of the box model are:

  • width
  • height
  • padding
  • border
  • margin

Width and height set the shape of an element. Padding sets the room between content and an element's border. Border sets the width, style and color of the border. Margin sets the room between elements.
