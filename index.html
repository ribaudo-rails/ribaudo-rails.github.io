Intro to game programming with JavaScript

In this course, we will explore programming concepts using the Phaser game development framework. We'll start off learning how to write JavaScripts for web browsers, then transition into the more complex framework of Phaser. Each week, new concepts will be introduced as tools to solve game design challenges. By the end of the 8 weeks, Ava and Stella should be comfortable using variables, loops and functions and both of them will have made their first interactive, web-based game.

Intro to Rails

In this course, we covered enough HTML and CSS to put together single-page sites with color and horizontal positioning. This CSS foundation put them in a good place to learn about Bootstrap, which the girls can use instead of writing all their CSS from scratch. We finished off the course by learning how to process web forms in Ruby on Rails to create simple mad libs and chatbot applications.
