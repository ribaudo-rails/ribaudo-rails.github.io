Today we went over centering and inline-block.

Centering

There are two ways to center content in HTML: centering inside and element and centering the actual element.

We centered content inside an element with:

text-align: center;

This is most useful for center text inside an element.

If we want to center an actual element, instead of what's inside that element, we can use:

margin-left: auto;
margin-right: auto;

This evens out the space on both the left and right sides of an element. This only works if an element is block-level and takes up less space than the width of the page.

Navbars

We created a navbar by changing the list items in the nav element from display:block; to:

display: inline-block;

This makes each element take up only as much space as it needs, instead of the full width of the page.
