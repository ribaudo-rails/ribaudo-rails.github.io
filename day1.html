We also talked about CSS. While HTML is used to structure content, CSS is used to style content. In other words, HTML deals with what content is and CSS deals with how it should look.

If we want to make an h1 red, we can do this with CSS

h1 {
  color: red;
}

This is called a CSS rule. It has a few different parts. The h1 is called the selector. It selects the HTML elements that will be affected. Then there are the { }. These curly braces hold all of the declarations in the rule. Each declaration usually takes up a full line. There's only one declaration in this rule. A declaration has two parts - a property and a value. color is the property and red is the value. They are separated by a : and each declaration ends with a ;
