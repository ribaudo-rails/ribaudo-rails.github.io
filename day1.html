Displaying a page in Rails

Starting a new Rails app

Use this command to create a Rails app:

rails new my_app

where my_app is your app's name.

Then, you need to go inside the app's directory in terminal (the black box at the bottom of the screen). cd stands for "change directory"

cd my_app

If you want to go back to the home directory, you can just use the command cd by itself.

Running a Rails app

Once you're inside your app's folder, you can use this command to start your rails server:

rails server

Then, click on "preview" in the top menu and select port 3000

Creating a route

All routes are stored in the routes file, which you can find at config > routes.rb

A route looks like this:

get "/my_form" => "application#form"

There are a few pieces to a route. "/myform" is the URL that the user types in the address bar to access this route. application is the controller name. We'll keep this the same for every route right now. form is the action name. It can be named whatever you want, but the action name should match up with an HTML file in the application folder.

Creating an HTML page

Web pages in Rails apps are called views. We can find them in app > views

First, we need to create a folder called application inside the views folder. This folder name needs to match up with the controller name from the route

Next, we need to create an HTML file called form.html inside the application folder. This filename needs to match up with the action name from the route.

Now, just write normal HTML code inside the file. Don't worry about the body or head elements - we'll talk about those later.
